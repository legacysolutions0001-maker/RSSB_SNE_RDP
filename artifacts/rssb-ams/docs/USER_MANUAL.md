# RSSB SNE AMS — User Manual

## Logging In
- Open the application
- Enter your Email and Password
- Click "Sign In"

## Dashboard
Shows live counts: Total Applicants, Today's Applicants, Male/Female/Transgender, Handicap, Couple/Single, Total Admins, and a table of the 10 most recent arrivals.

## Adding an Applicant
1. Click "Add Applicant" in the sidebar
2. Fill in all required fields
3. Click "Save Applicant"

## Searching Applicants
- Use the search bar on the Applicants page to search by name, phone, or centre
- Use the filter dropdowns for Gender, Couple/Single, Handicap

## Exporting Reports
1. Go to "Reports" in the sidebar
2. Apply filters as needed
3. Click "Export Excel" or "Export PDF"

## Taking a Backup
1. Go to "Backup" in the sidebar
2. Click "Create Backup"
3. A JSON file will be downloaded to your computer
4. Store it safely

## Restoring from Backup
1. Go to "Backup" in the sidebar
2. Click "Restore Backup"
3. Select your backup JSON file
4. Confirm the restore

## Managing Admins (Super Admin only)
1. Go to "Admins" in the sidebar
2. Click "Create Admin" to add a new admin
3. Use the action buttons to Edit, Disable, or Delete an admin
